# Customer-Behavior-Sales-Analysis

Customer Behavior & Sales Insights Analysis is a data analytics project focused on analyzing customer purchasing behavior, spending patterns, and marketing campaign performance. The project aims to understand how factors such as age, income, education level, marital status, and product preferences influence customer spending and engagement. The dataset was analyzed using Microsoft Excel, where data cleaning, transformation, and visualization techniques were applied to generate meaningful business insights. Interactive dashboards with KPI cards, pivot tables, charts, and slicers were created to visualize customer trends and sales patterns in a clear and user-friendly manner. This project highlights customer segmentation, purchasing behavior, and campaign effectiveness while demonstrating practical skills in data analysis, dashboard creation, and business insight reporting.
